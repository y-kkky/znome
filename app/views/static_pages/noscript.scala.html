Для використання сайту у вас повинен бути включений javascript.

Internet Explorer 9

У меню Знаряддя (Tools) виберіть пункт Властивості браузера (Internet Options).

Відкрийте вкладку Security (Безпека) та виберіть зону Інтернет (Internet). Натисніть на кнопку Інший (Custom level).

У розділі Сценарії (Scripting), у підрозділах Активні сценарії (Active scripting) та Виконувати сценарії аплетів Java (Scrtpting of Java applets ) виберіть пункти Увімкнути (Enable).

Firefox 7.0

У меню Інструменти (Tools) виберіть пункт Налаштування (Options).

Перейдіть у розділ Вміст (Content) і встановіть прапорець поряд із Дозволити JavaScript (Enable JavaScript).

Opera 11.х

У меню Інструменти (Tools) виберіть пункт Налаштування (Preferences).

Перейдіть на вкладку Додатково (Advanced). Виберіть пункт меню Вміст (Content) і встановіть прапорець поряд із Увімкнути JavaScript (Enable JavaScript).
